SmartHealth Connect System

 Overview

SmartHealth Connect System is a healthcare management system designed to improve communication between patients, doctors, and administrators. The system provides an easy-to-use platform for appointment scheduling, medical record management, diagnosis updates, and healthcare notifications.

The project aims to replace manual healthcare processes with a secure and efficient digital solution.
---

 Objectives

- Digitize healthcare services.
- Simplify appointment scheduling.
- Maintain secure electronic medical records.
- Improve communication between patients and doctors.
- Reduce paperwork and administrative workload.

---

 Features

 Patient
- Register an account
- Login securely
- Book appointments
- View medical records
- Receive notifications
- Cancel appointments

### Doctor
- Login
- View appointments
- Access patient records
- Upload diagnoses
- Prescribe treatments

### Administrator
- Manage users
- Manage the system
- Generate reports
- Monitor system activities

## Installation

1. Install Python 3.x.
2. Install XAMPP and start Apache and MySQL.
3. Import the SQL database into phpMyAdmin.
4. Install the required Python packages.
5. Run the main application.

Example:

```bash
python main.py
```
